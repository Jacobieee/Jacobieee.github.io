  1. Remove Duplicates from Sorted Array

    Given a sorted array nums, remove the duplicates in-place such that each element appear only once and return the new length.

    Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1) extra memory.

There are 2 things that we need to notice.

  • The array needs to be changed in-place.
  • No extra memory is allowed to be allocated.

So, we use 2 pointers to solve it. One goes through the old array and the other one goes through the new one. Once the first pointer meets a new number, the second pointer adds this number in and moves to the next place.
